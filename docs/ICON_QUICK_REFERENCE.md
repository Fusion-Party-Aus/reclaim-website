## How to Use

1. Copy the icon name from the middle column
2. Paste into Sanity's Icon field
3. Make sure it includes the `mdi:` prefix

**Example:**

```
For a housing policy, enter: mdi:home-city
For a transport policy, enter: mdi:train
```

## Search Tips

When searching on https://pictogrammers.com/library/mdi/:

- Try different keywords: "house" vs "home", "hospital" vs "medical"
- Look for outline versions: add "-outline" to icon name
- Try variations: "train", "subway", "railway", "metro"
- Check related icons: The site shows similar icons

## Icon Naming Convention

MDI icons follow patterns:

- Base noun: `home`, `train`, `hospital`
- With modifier: `home-city`, `train-car`, `hospital-box`
- With action: `home-remove`, `currency-usd-off`, `account-plus`
- Outline style: `home-outline`, `heart-outline`
