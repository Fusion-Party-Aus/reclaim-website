Usage Guidelines

Best practices for implementing the design system

Do's

  • ‚úÖ Use high contrast color combinations for accessibility
  • ‚úÖ Maintain consistent spacing using the scale
  • ‚úÖ Use bold typography for emphasis and hierarchy
  • ‚úÖ Leverage gradients sparingly for visual interest
  • ‚úÖ Keep button labels action-oriented and clear
  • ‚úÖ Use emoji icons for personality and quick recognition

Don'ts

  • ‚ùå Don't use low-contrast color combinations
  • ‚ùå Don't create custom spacing values
  • ‚ùå Don't use more than 3 font sizes per section
  • ‚ùå Don't overuse gradients or animations
  • ‚ùå Don't use vague button labels like "Click here"
  • ‚ùå Don't mix icon styles (stick to emoji)

Implementation

Quick start with CSS custom properties

/* Include in your stylesheet */ :root { --magenta: #C926F2; --mint: #5EFFD8; --lavender: #9A94E7; --black: #010102; --white: #FFFFFF; --space-sm: 1rem; --space-md: 1.5rem; --space-lg: 2rem; --space-xl: 3rem; --radius-sm: 4px; --radius-md: 8px; --transition-base: 0.3s ease; } /* Use in your components */ .button { background: var(--magenta); padding: var(--space-md) var(--space-lg); border-radius: var(--radius-sm); transition: all var(--transition-base); }
